# LProctor
Advanced Chat Moderation &amp; Security System for Minecraft 

# Under Development
LProctor acts as the guardian of your community. It actively monitors, filters, and protects your server chat from toxic behavior and spam.

Project Status: Active Development

Goal: Providing a safe, toxic-free environment for players.

Expect the first build to be available soon!
